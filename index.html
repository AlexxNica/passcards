passcards aims to be a simple, cross-platform (desktop and mobile) password manager that is compatible with 1Password. Available as a browser add-on, a web app and as a command-line app.

Current Status

  • The command line version of passcards can show, add, edit and remove items from a 1Password vault. It can also create new vaults and generate passwords for items.
  • The web and browser add-ons can list items in an existing 1Password vault that has been synced to Dropbox and autofill username/password fields. There is no support yet for creating new vaults, adding, editing or removing logins.

Get passcards

Firefox add-on

For Firefox users, passcards is available as an add-on for your browser. Download latest version.

Chrome extension

For Chrome users, the extension can be installed from the Chrome Web Store.

Web app

The web-based version of passcards can be accessed here. Currently supports Firefox, Google Chrome and Safari.

Support for Internet Explorer 10 and other modern browsers is planned.

Command Line

The passcards command-line app can be installed using npm. 
	npm install -g passcards
