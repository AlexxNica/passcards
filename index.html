A simple, open source password manager. Available as a browser add-on, a web app and as a command-line app.

Firefox add-on

For Firefox users, passcards is available as an add-on for your browser. Download latest version.

Web app

The web-based version of passcards can be accessed here. Currently supports Firefox, Google Chrome and Safari.

Support for Internet Explorer 10 and other modern browsers is planned.

Command Line

The passcards command-line app can be installed using npm. 
npm install -g passcards
